A menu-driven Library Management System developed in C++ using Object-Oriented Programming (OOP) and File Handling. The application helps manage library books, search records, and handle book issue/return operations efficiently.

##Features:- 
- Add New Books
- Display Available Books
- Search Books by Title
- Search Books by Author
- Issue Books
- Return Books
- Persistent Data Storage using File Handling

##Technologies Used:
- C++
- Object-Oriented Programming (OOP)
- File Handling (fstream)

##Concepts Implemented:
- Classes and Objects
- Functions
- Loops
- Conditional Statements
- File Operations
- Record Management
